# SocialNetworkApp
Developed in Java using Java FX for graphical user interface. Using sql-postgre for database connection.

Features:
1. Log in
2. Sign up
3. Forgot Password
4. Search user
5. See the profile of an user
6. Send a friend request to an user
7. Accept/delete a friend request
8. Log out
